# Build
```
go build .
```
# Usage
```
Usage: ip netns exec ns<wgi> ./endpoint-vpn-usage-stats
  -debug
        print errors to stderr, indented json output
  -wgi string
        wg interface, e.g. wg0, required
  -accel-cmd
        accel-cmd data required
```
